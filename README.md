Project description:
Drinkifyyy is a mini-application for discovering and viewing cocktail recipes. Users can search cocktails by name, browse categories, open detailed modals, switch themes, and add cocktails to Favorites using localStorage. The application includes a responsive layout, pagination, and dynamic rendering of data fetched from a remote API. Built fully with clean, modular structure and reusable components.

Tools/Technology:
HTML5, CSS3, JavaScript (ES6+), REST API, TUI Pagination, LocalStorage, Responsive Design
